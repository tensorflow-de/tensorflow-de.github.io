Keras ist eine Open Source Deep-Learning-Bibliothek, geschrieben in Python.

Sie wurde von François Chollet initiiert und erstmals am 28. März 2015 veröffentlicht.

Das erklärte Ziel von Keras ist es, eine vereinfachte API für eine Vielzahl von Deep Learning Problemstellungen zu stellen, ohne dass sich der Data Scientist mit den zugrundeliegenden mathematischen Details beschäftigen muss.

Seit dem Release von TensorFlow 1.4 ist Keras Teil der Tensorflow Core API und seit dem Release von TensorFlow 2.0 ist Keras die bevorzugte API.
